JSX(javascript and XML) ，他是 JavaScrip 的一种扩展语法，让 JavaScript 语言能够支持这种直接在 JavaScript 代码里面编写类似 HTML 标签结构的语法。
通过 babel 编译 会把类似 HTML 的 JSX 结构转换成 JavaScript 的对象结构。JSX 可能会让你想起某种模板语言，但是它具有 JavaScrip 的全部能力。
JSX 只是调用 React.createElement(component, props, ...children) 的语法糖。 但是更加便捷。
